# 🦁 Zoo Quiz Bot — Telegram-бот "Твоё тотемное животное"

**Домашняя работа** по курсу разработки Telegram-ботов.  
Бот-викторина для Московского зоопарка, помогающий пользователям выбрать тотемное животное и узнать о программе опеки.

---

## 📌 Функциональность

| Раздел | Описание |
|--------|----------|
| **🎮 Викторина** | 5 вопросов с вариантами ответов. На основе ответов пользователь получает одно из 10 тотемных животных. |
| **🦊 Животные** | Амурский тигр, белый медведь, ворон, сова, слон, выдра, капибара, жираф, шимпанзе, волк. У каждого — описание, интересный факт и информация об опеке. |
| **📸 Результат** | Фото животного + персонализированное описание. |
| **📞 Контакт с сотрудником** | Отправка уведомления на email с результатами пользователя. |
| **📤 Поделиться** | Готовый текст для соцсетей со ссылкой на бота. |
| **⭐ Обратная связь** | Сбор отзывов пользователей в базу данных. |
| **📊 Статистика** | Команда `/stats` для администраторов (просмотр метрик). |
| **🔄 Перезапуск** | Кнопка «Заново» в любой момент. |

---

## 🛠 Технологии

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.12 |
| Библиотека | python-telegram-bot 20.7 |
| База данных | SQLite |
| Почта | aiosmtplib (Yandex SMTP) |
| Логирование | RotatingFileHandler |
| Мониторинг | Кастомный модуль с декораторами |

# Настройка .env

text
BOT_TOKEN=ваш_токен
ADMIN_IDS=ваш_telegram_id
